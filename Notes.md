# StashStats Notes

## Packages

- *dash*: Core framework for building web applications with pure Python
- *plotly*: Interactive graphing library that Dash uses for visualizations
- *sqlalchemy*: SQL toolkit and Object-Relational Mapping (ORM) library for database interactions
- *pydantic*: Data validation library using Python type annotations
- *python-dotenv*: Loads environment variables from .env files
- *requests*: HTTP library for making API calls and web requests
- *psycopg2-binary*: PostgreSQL database adapter for Python
- *pandas*: Data manipulation and analysis library
- *dash-bootstrap-components*: Bootstrap components for styling Dash apps
